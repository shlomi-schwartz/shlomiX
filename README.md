# shlomiX

Claude Code plugin marketplace by [shlomi-schwartz](https://github.com/shlomi-schwartz).

## Install the marketplace

In Claude Code:

```
/plugin marketplace add shlomi-schwartz/shlomiX
```

## Available plugins

| Plugin | Description |
|--------|-------------|
| **ux-expert** | Always-on UX advisor powered by 30 Laws of UX |

### Install a plugin

```
/plugin install ux-expert@shlomiX
```
